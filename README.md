## Description

Application web développée en Python permettant la collecte de données
sur les habitudes numériques des étudiants, suivie d'une analyse
descriptive automatique des résultats.

## Fonctionnalités

- Collecte via formulaire structuré (18 questions)
- Stockage persistant des réponses (base de données SQLite)
- Tableau de bord avec visualisations interactives
- Analyse descriptive : moyenne, médiane, écart-type, quartiles
- Matrice de corrélation entre variables
- Export des données au format CSV

## Technologies utilisées

- Python 3.11
- Streamlit — interface web
- Pandas — traitement des données
- Plotly — visualisations
- SQLite — base de données

## Lancer l'application en local

```bash
pip install -r requirements.txt
streamlit run app.py
```
